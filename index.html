ABOUT
ME

에듀테크 도메인의 저작도구와 콘텐츠 허브 솔루션을 교육청과 학교 등 공공기관 대상 서비스로 구축하는 프로젝트에서 웹 접근성 기준을 충족하는 UI를 설계하고 구현 해왔습니다.

교육 데이터 기반의 대시보드와 리포트 UI를 구축하며 ApexCharts, Chart.js를 활용한 데이터 시각화 작업에 능숙 하며, 사용자가 데이터를 직관적으로 이해할 수 있도록 돕는 인터랙티브 UI 에 관심이 있습니다.

다양한 디바이스(PC, 태블릿, 크롬북, 모바일 웹뷰)와 브라우저(Chrome, Safari, Firefox, Whale) 환경에서의 대응 경험을 바탕으로, 플랫폼 간 차이를 고려한 안정적인 UI를 만들고 이슈를 해결 할 수 있습니다.

05. THE CAMP 메인 페이지 리뉴얼

다수의 개발자가 접근성 이슈 없이 빠르게 매핑 작업을 할 수 있도록 프로젝트에 사용된 공통 컴포넌트를 정리한 가이드 시스템을 제공했습니다. 명확한 사용 규칙 정립을 통해 UI 일관성을 확보하고, 불필요한 리소스 낭비를 줄여  프로젝트의 완성도를 높였습니다.

교사, 학생, 학부모 등 다양한 사용자 층을 고려하여  PC, 크롬북, 태블릿, 모바일을 아우르는 Full 반응형을 구현했습니다. SCSS 루프와 @mixin을 활용해 테마별 CSS 변수를 자동 생성하는 체계를 구축하고, 코드 수정 없이 컬러셋 교체만으로 신규 테마 확장이 가능한 환경을 구성했습니다.

사용자 정의 테마 적용 시 컴포넌트 State 컬러가  일관되게 적용되지 않는 문제를 Primary 색상을 기준으로 HSL 기반 상태 컬러 자동 생성 로직을 구현했습니다.  HSL의 Lightness값을 조절하여 State 컬러를 자동 생성하여 웹 접근성 4.5:1 대비 이상 유지하도록 작업했습니다.

Framer Motion 스크롤 애니메이션을 활용하여 단순한 정보 전달을 넘어,  사용자의 행동에 반응하는 인터랙션을 구현했습니다. 차트 라이브러리를 활용하여 디자인 요구사항에 맞게 커스터마이징하여 심리 검사 레포트를 작업했습니다.

iOS WebView 환경에서 Appbar 드래그 시 하단 공백이 노출되고, iOS에서만 Notch 영역 높이가 다르게 적용되어 플랫폼별 UI 경험이 달라지는 문제를 개선했습니다.
